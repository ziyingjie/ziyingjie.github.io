基于jQuery，所以必须在之前引入jQuery

navToggle

  • 插件功能：实现导航栏在滚动过程中的显示隐藏特效

  • 传入参数为对象，包含4个属性：

  • selector: 选择器 必须
  • animate: 动画方式 可选
  • time: 过渡时间 可选
  • distance: 执行动画的距离 可选
  • 选择器为必须传入的参数, 其余参数可选，其中动画方式可选”fade”,”slide”

  • options = { selector, animate, time, distance }
